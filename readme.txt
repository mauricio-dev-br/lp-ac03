A terceira atividade contempla a criação e utilização de módulos do python (através do import), o levantamento/lançamento de exceções (não o tratamento, que será visto apenas na unidade 4) e a interação entre objetos e classes diferentes. Para isso vocês devem implementar as classes dos arquivos em anexo de acordo com as instruções dadas na string de documentação de cada classe/método.

### ATENÇÃO ###
Devem ser entregues exatamente 3 arquivos *.py com exatamente os mesmos nomes dos arquivos anexos a esta atividade.
* cliente.py
* conta.py
* banco.py

Para esta atividade serão aplicados os seguintes descontos na nota:
- 1 ponto para código executado durante a importação do módulo;
- 0,5 pontos para cada arquivo com nome diferente do pedido;
- 0,5 pontos para cada arquivo entregue a mais, além dos 3 arquivos pedidos.

A entrega deve ser feita aqui pelo próprio Classroom, anexando os arquivos à atividade e marcando-a como entregue. Não serão aceitas entregas feitas por quaisquer outros meios (email, git, etc.) e não serão aceitos nenhum outro formato de arquivo (pdf, jpg, txt, zip, etc.)

A correção será feita utilizando testes automatizados, portanto atentem-se às regras:
- Não alterar o nome do arquivo (serão descontados pontos para nomes diferentes, como explicado acima e se houver erros de importação devido a diferenças nos nomes do arquivo, isso pode acarretar em falha nos testes de correção);
- Entregar um único arquivo executável do python (extensão *.py - serão descontados pontos para arquivos entregues que não tenham sido pedidos.)
- Preencher com o seu email Impacta o cabeçalho do arquivo, lembrando que a linha do email deve estar comentada.
- Arquivos com erro de sintaxe não serão corrigidos.
